## Getting Started

1. **Prerequisites**: Ensure you have the AWS CLI installed and configured with your credentials.

2. **Run the Project**:
   ```bash
   ./start.sh
   ```
   This will display an overview of all available scripts.

3. **Using Individual Scripts**:
   Each script accepts a bucket name as an argument. Example:
   ```bash
   ./s3/bash-scripts/list-bucket my-bucket-name
   ./s3/bash-scripts/put-object my-bucket-name /path/to/file
   ```

## Scripts Overview

- **create-bucket**: Creates a new S3 bucket
- **delete-bucket**: Deletes an existing S3 bucket
- **list-bucket**: Lists all S3 buckets in your account
- **list-object**: Lists all objects in a specified bucket
- **put-object**: Uploads files to an S3 bucket
- **sync**: Synchronizes local directories with S3 buckets

## Purpose

This project serves as a learning repository for AWS SAA-C03 exam preparation, providing practical examples of common S3 operations using the AWS CLI.
